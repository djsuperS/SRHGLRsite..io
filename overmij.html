Ik vind mijn VMBO opleiding erg fijn omdat ik een leuke vriendengroep heb op school maar ook omdat ik een flex rooster heb.

bij bijvoorbeeld vakken waarbij je nog iets niet snapt of waarvoor je met andere leerlingen samenwerkt aan een project.

Mijn programmeer ervaring

Ik heb al eerder een paar dingen gemaakt met Python waaronder een foto herkennings script dat een reactie of toets indrukte bij
het zien van een foto. Ook heb ik een soort stem herkennings programma gemaakt dat reageerde op dingen zoals "hallo hoe gaat het"
en een paar andere simpele vragen. Het foto herkennings programma was gemaakt met de library: pyautogui. En voor het stem herkennings
programma gebruikte ik google speech recognition. Ik heb een groot deel van de HTML cursus van codecademy
gedaan, hierdoor weet ik dus wel al een beetje maar toch had ik nog veel vragen bij het maken van deze site.
